Timestamp Microservice

How to use?

Simply type in a unix timestamp or a natural language date after the api web address like such: (use % as spaces)
https://fcc-api-development-dacuban.c9users.io/December%2015,%202015
or
https://fcc-api-development-dacuban.c9users.io/1450137600

Sample Response:

{ "unix": 1450137600, "natural": "December 15, 2015" }
